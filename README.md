# arunbhatia.dev

My personal portfolio site. Built with React and Three.js.

Live at: https://arunbhatia-dev.github.io

## Running locally

```bash
npm install
npm start
```

Opens at localhost:3000

## Deploy

```bash
npm run deploy
```

Builds and pushes to gh-pages.

## Stack

- React + TypeScript
- Three.js / React Three Fiber (3D laptop)
- Plain CSS
